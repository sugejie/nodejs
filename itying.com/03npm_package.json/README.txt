主要内容
    1.npm安装第三方模块模块
        1.1:npm install module_name         //安装第三方模块，
        1.2:npm install module_name --save  //写入package.json
    2.package.json
        2.1:npm init --yes  //生成package.json,项目名要合法
        2.2:npm install     //自动从package.json下载依赖包
    3.cnpm淘宝镜像
        - npm网址：https://www.npmjs.com
        - cnpm网址：https://npm.taobao.org
        安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org
